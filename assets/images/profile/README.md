# Profile Images

Place your profile photo here.

## Recommended Specifications

- **Format**: JPG or PNG
- **Size**: 500x500px (square)
- **File size**: Under 500KB
- **Filename example**: `profile.jpg`

## How to Use

1. Add your photo to this folder
2. Update `js/config.js`:
   ```javascript
   profileImage: "assets/images/profile/your-photo.jpg"
   ```

## Image Optimization Tips

- Use [TinyPNG](https://tinypng.com) to compress images
- Crop to square aspect ratio
- Use high quality but compressed files
- Recommended: 72 DPI for web
