# Project Images

Place your project screenshots and images here.

## Recommended Specifications

- **Format**: JPG or PNG
- **Size**: 1200x630px (landscape) or 800x800px (square)
- **File size**: Under 1MB each
- **Filename**: Use descriptive names (e.g., `apex-dashboard.jpg`, `truck-simulator-screenshot.png`)

## How to Use

1. Add your project images to this folder
2. Update the project in `js/config.js`:
   ```javascript
   projects: [
     {
       name: "My Project",
       image: "assets/images/projects/my-project.jpg",
       // ... other fields
     }
   ]
   ```

## Image Optimization Tips

- Use [TinyPNG](https://tinypng.com) to compress images
- Take high-quality screenshots
- Show the most interesting part of your project
- Use 16:9 or 1:1 aspect ratio
- Recommended: 72 DPI for web

## Examples

- `apex-project.jpg` - Screenshot of ApeX platform
- `truck-simulator.png` - Truck simulator interface
- `lidar-backpack.jpg` - LiDAR device photo
- `messenger-app.png` - Messenger GUI screenshot
