# Images Folder

Place your images here for the website.

## Recommended Sizes

### Profile Picture
- **Filename:** `profile.jpg` or `profile.png`
- **Dimensions:** 800x800px (square)
- **Format:** JPG or PNG
- **Size:** < 500KB

### Project Screenshots
- **Naming:** `projectname-screenshot.jpg` (e.g., `apex-screenshot.jpg`)
- **Dimensions:** 1200x800px (landscape) or 800x800px (square)
- **Format:** JPG or PNG
- **Size:** < 1MB each

### Open Graph Image (Social Media Sharing)
- **Filename:** `og-image.jpg`
- **Dimensions:** 1200x630px
- **Format:** JPG
- **Size:** < 1MB

## Tips

1. **Compress images** before uploading using [TinyPNG](https://tinypng.com/)
2. **Use descriptive names** like `truck-simulator-demo.jpg`
3. **Avoid spaces** in filenames (use hyphens instead)
4. **Keep file sizes small** for faster loading
